## Overview

A collection of focused CLI tools that automate common friction points in the development lifecycle. Each tool is independent, composable, and designed around a specific workflow rather than trying to be a monolithic platform.

## Tools

- **Scaffold**: Project generator with template support and interactive prompts.
- **EnvManager**: Environment variable management with validation and secret handling.
- **DepAudit**: Dependency graph analysis with upgrade impact reporting.
- **PreviewDeploy**: One-command preview environment provisioning.

## Philosophy

Each tool prioritizes simplicity over configurability. Defaults are opinionated but overridable. The goal is to reduce the number of decisions developers need to make before they can start building.
