## Overview

Created a reusable design system toolkit for building editorial-style web experiences. The system includes a comprehensive token architecture for colors, typography, spacing, and motion — plus ready-to-use React components with built-in animation hooks.

## Design Approach

The token system is the foundation: all visual decisions flow from a single source of truth. Components are intentionally minimal — the design system provides structure, not decoration. Animation primitives use GSAP ScrollTrigger for scroll-driven reveals without framework lock-in.

## Technical Stack

- **Styling:** Tailwind CSS v4 with custom tokens
- **Animation:** GSAP with ScrollTrigger plugin
- **Framework:** React 18 with TypeScript
- **Documentation:** MDX-based storybook
- **Build:** Vite with library mode

## Key Learnings

Design tokens need to be framework-agnostic at the definition level, with adapter layers for each target (CSS variables, Tailwind config, Figma tokens). The biggest challenge was maintaining consistency across three parallel token representations.
